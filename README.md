# 2026Spring-Team05-Lavoine

## The Project
**Sustainable Box Trivia Game** (*Proposed and sponsored by Dr. Nathalie Lavoine*)

The science of sustainable packaging goes far beyond 'just a box', but sadly most people remain uninformed and uninterested about the subject.
To improve education in an enjoyable and engaging fashion, we are creating a fun learning tool in the form of a packaging-themed teacher-hosted trivia game!

The project is a full-stack application to host the game. Key features include:
- a teacher dashboard with ways to add topic categories, trivia questions/answers, and student users
- a way to host live games with many (20-30) students
- a 'teaching' gamemode for assessment
- a 'multiplayer' gamemode for fun competition
- a 'solo' mode for individual study
- an interactive 3d box-themed visualization of a student's progress in topic categories

## The Tech
The project is developed with the following technologies:
- Java as the backend language
- MySQL for the database
- React JS for the frontend
- we are exploring frameworks and libraries for backend-frontend and backend-database communication, 3d visualization, and live networked gameplay
